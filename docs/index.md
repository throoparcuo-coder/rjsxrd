## Краткий обзор

rjsxrd - это автоматически обновляемая коллекция публичных VPN-конфигураций для обхода блокировок.

**Основная информация:**
- Обновление: раз в 2 дня (00:00 UTC) через GitHub Actions
- Протоколы: VLESS, VMess, Trojan, Shadowsocks, Hysteria2, TUIC
- Верификация: Xray-core батч-тестирование с сортировкой по пингу
- Особенности: SNI/CIDR фильтрация, Telegram прокси, разделение файлов по 300 конфигов

**Полная документация:** см. [overview.md](overview.md)

Для начала работы рекомендуется ознакомиться с разделом [Установка и использование](installation.md).
